# javascript-web-chart

## Description
A small project to demonstrate using
client side scripting for metrics.

## Tech stack
- jquary 3.5
- chart.js 2.9.4

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`
Availble at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
